# Ada 2022 Documents Overview

This page from the Ada Resource Association provides access to official documentation for Ada 2022, the latest revision of the Ada programming language standard.

## Key Documents Available

**Ada 2022 Reference Manual**: The primary standard document corresponding to ISO/IEC 8652:2023(E), available for online browsing and download in multiple formats including PDF and HTML archives.

**Annotated Reference Manual (AARM)**: Intended for compiler developers and standards experts, this version includes detailed annotations explaining the language specification's technical details.

**Ada 2022 Overview**: An unofficial guide by Jeff Cousins that explains new features, provides usage examples, and discusses compatibility with earlier Ada versions (95, 2005, and 2012).

## Download Options

The materials are offered in various formats to suit different needs:
- PDF versions with highlighted changes or change bars
- HTML files for online reference
- Text-only versions for accessibility

The page also references additional resources and tools available through www.ada-auth.org, the standards authority website.
